What is the VWB API?
==============

The Voting With Blockchain API is a RESTful API created with sole intention of providing a way to
take vote management into your own hands! Our API acts as a middleware that handle the creation
of and maintenance of elections, storing information of voters and candidates, and the transportation
of votes.
